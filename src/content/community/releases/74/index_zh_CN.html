NetBeans IDE 7.4 信息

NetBeans IDE 7.4 将 NetBeans IDE 7.3 中引入的高级 HTML5 开发支持扩展到了 Java EE 和 PHP 应用程序开发中，同时为 Android 和 iOS 平台上的混合 HTML5 应用程序开发提供了新的支持。另外，此发行版为使用 JDK 8 的预览版本提供了支持，并包括对 JavaFX、C/C++ 等的持续改进。

NetBeans IDE 7.4 提供了英语、巴西葡萄牙语、日语、俄语和简体中文版本。

新增功能集锦

HTML5

  • Cordova 应用程序开发
  • Android 和 iOS 浏览器支持
  • 对以下样式表语言的编辑支持：SASS 和 LESS
  • 主工具栏中的浏览器切换程序
  • 保存对 Chrome Developer Tools 进行的更改
  • 网络监视器
完整屏幕快照

JavaScript

  • 对 AngularJS、Knockout 和 ExtJS 框架的编辑支持
  • JSON 文件中的导航器和代码折叠
  • 增强的代码完成功能，准确性更高
完整屏幕快照

Java EE

  • Java EE 项目中可用的 HTML5 功能
  • 用于 JSF 2.2 资源库合同和 FacesComponent 的向导

完整屏幕快照

PHP

  • PHP 应用程序中可用的 HTML5 功能
  • Nette 框架 2（带有 Latte 模板）和 Zend 框架 2 支持
  • Atoum 测试框架支持
  • 编辑器和重命名类型重构改进
  • 静态代码分析支持
完整屏幕快照

Java

  • JDK 8 功能的预览支持：配置文件，Lambda
  • 代码完成、提示和重构改进 
  • 本机打包
  • 更新了捆绑的 Ant 1.9.1 和 Maven 3.0.5

JavaFX

  • Maven 项目中的 "New FXML File"（新建 FXML 文件）对话框
  • 更紧密地结合 JavaSE 和 JavaFX 项目类型
  • 对项目部署选项的改进

C/C++

  • 运行/调试启动程序支持
  • 查找用例改进
  • 每个项目的 C/C++ 格式样式

其他

  • "New Tasks Dashboard and Notifications"（新任务显示板和通知）窗口
  • 性能分析任务的锁争用
  • Bugzilla 脱机模式
  • 对版本控制工具的增强：Subversion，Git，Mercurial
  • TestNG 和 Junit 改进
  • 对数据库的改进

有关此发行版本功能的详细信息，请参见 NetBeans IDE 7.4 新增功能和重要功能页。
