Informações do NetBeans IDE 7.4

O NetBeans IDE 7.4 estende o suporte ao desenvolvimento do HTML5 apresentado no NetBeans IDE 7.3 para o desenvolvimento de aplicações Java EE e PHP, ao mesmo tempo que oferece um novo suporte para o desenvolvimento de aplicações HTML5 híbridas nas plataformas Android e iOS. Além disso, essa release oferece suporte para trabalhar com versões de visualização de JDK 8 e inclui aprimoramentos contínuos para JavaFX, C/C++ e muito mais.

O NetBeans IDE 7.4 está disponível em inglês, português do Brasil, japonês, russo e chinês simplificado.

Novas Funcionalidades em Destaque

HTML5

  • Desenvolvimento da aplicação Cordova
  • Suporte para Browsers Android e iOS
  • Suporte de edição para linguagens de folha de estilo: SASS e LESS
  • Switcher do browser na barra de ferramentas principal
  • Salvando alterações das Ferramentas de Desenvolvedor do Chrome
  • Monitor de rede

JavaScript

  • Editando o suporte para frameworks do AngularJS, do Knockout e do ExtJS
  • Navegador e dobramento de código em arquivos JSON
  • Conclusão de código aprimorada com precisão melhorada

Java EE

  • Funcionalidades HTML5 disponíveis nos projetos Java EE
  • Assistentes de Contratos da Biblioteca de Recursos e Componentes de Faces do JSF 2.2

PHP

  • Funcionalidades de HTML5 disponíveis nas aplicações PHP
  • Suporte a Nette Framework 2 (com modelos Latte) e Zend Framework 2
  • Suporte a framework de teste do Atoum
  • Melhorias na refatoração do tipo de renomeação e do editor
  • Suporte à análise de código estático
