NetBeans IDE 7.4情報

NetBeans IDE 7.4では、AndroidおよびiOSプラットフォームにおけるハイブリッドHTML5アプリケーションの開発へのサポートが提供されるとともに、NetBeans IDE 7.3で導入された拡張HTML5の開発サポートをJava EEおよびPHPアプリケーションの開発まで拡張されています。また、このリリースでは、プレビュー・バージョンのJDK 8を使用した作業のサポートが提供され、JavaFX、C/C++などに対する継続的な機能拡張も含まれています。

NetBeans IDE 7.4は、英語、ポルトガル語(ブラジル)、日本語、ロシア語および簡体字中国語で使用できます。

新機能の概要

HTML5

  • Cordovaアプリケーションの開発
  • AndroidおよびiOSブラウザのサポート
  • スタイルシート言語の編集サポート: SASSおよびLESS
  • メイン・ツールバーのブラウザ・スイッチャ
  • Chrome開発者向けツールからの変更の保存
  • ネットワーク・モニター
フル・スクリーンショット

JavaScript

  • AngularJS、KnockoutおよびExtJSフレームワークの編集サポート
  • JSONファイルでのナビゲータおよびコード折りたたみ
  • 精度が改善された拡張コード補完
フル・スクリーンショット

Java EE

  • Java EEプロジェクトでHTML5の機能を使用可能
  • JSF 2.2 Resource Library ContractsおよびFacesComponentsのウィザード

フル・スクリーンショット

PHP

  • PHPアプリケーションでHTML5の機能を使用可能
  • Nette Framework 2 (Latteテンプレートを含む)およびZend Framework 2のサポート
  • Atoumテスト・フレームワークのサポート
  • エディタおよび名前変更タイプ・リファクタリングの改善
  • 静的コード分析のサポート
フル・スクリーンショット

Java

  • JDK 8の機能のプレビュー・サポート: Profile、Lambda
  • コード補完,、ヒントおよびリファクタリングの改善 
  • ネイティブ・パッケージ
  • 更新されたバンドルAnt 1.9.1およびMaven 3.0.5

JavaFX

  • Mavenプロジェクトでの新規FXMLファイル・ダイアログ
  • JavaSEとJavaFXのプロジェクト・タイプのより緊密な整合
  • プロジェクト・デプロイメント・オプションの改善

C/C++

  • 実行/デバッグ・ランチャのサポート
  • 使用方法の検索の改善
  • プロジェクトごとのC/C++書式設定スタイル

その他

  • 新規タスク・ダッシュボードおよび通知ウィンドウ
  • プロファイリング・タスクのロックの競合
  • Bugzillaオフライン・モード
  • バージョニング・ツールへの機能強化: Subversion、Git、Mercurial
  • TestNGおよびJunitの改善
  • データベースの改善

このリリースの機能の詳細は、NetBeans IDE 7.4の新機能と注目すべき機能のページを参照してください。
