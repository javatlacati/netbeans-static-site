Informações do NetBeans IDE 8.1

O NetBeans IDE 8.1 fornece analisadores de código e editores prontos para uso para trabalhar com as tecnologias Java 8 mais recentes -- Java SE 8, Java SE Embedded 8 e Java ME Embedded 8. O IDE também tem uma variedade de novas ferramentas para HTML5/JavaScript, especialmente para Node.js, KnockoutJS e AngularJS; aprimoramentos que melhoram ainda mais seu suporte para Maven e Java EE com o PrimeFaces e aprimoramentos para o suporte a PHP e C/C++.

O NetBeans IDE 8.1 está disponível em inglês, português do Brasil, japonês, russo e chinês simplificado.

Novo Projeto e Melhorias do Criador de Perfil

  • Interface do usuário reprojetada
  • Novas funcionalidades:
    • Árvores de encaminhamento e reversão dinâmica de chamadas nos resultados da CPU
    • Árvores de alocação dinâmica nos resultados da Memória
    • Monitorando a utilização da CPU
    • Dumps de threads da aplicação com perfil
  • Aperfeiçoamentos no mecanismo:
    • Conexão mais rápida para uma aplicação com perfil
    • Chamadas de saída limitadas de métodos com perfil
    • Criação de perfil de memória para classes selecionadas
  • Configuração simplificada do criador de perfil
  • Integração aperfeiçoada no IDE
