NetBeans IDE 8.1 信息

NetBeans IDE 8.1 提供了现成的代码分析器和编辑器，可在其中使用最新的 Java 8 技术 -- Java SE 8、Java SE 嵌入式 8 和 Java ME 嵌入式 8。IDE 还提供一系列面向 HTML5/JavaScript 的新工具，尤其适用于 Node.js、KnockoutJS 和 AngularJS；能够进一步改进对 Maven 以及带 PrimeFaces 的 Java EE 的支持的增强功能；以及针对 PHP 和 C/C++ 支持的改进。

NetBeans IDE 8.1 提供了英语、巴西葡萄牙语、日语、俄语和简体中文版本。
